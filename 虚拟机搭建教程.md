# 🖥️ Windows 和 macOS 虚拟机搭建教程

**2025年9月6日发布。**

本教程介绍如何在 **Windows 系统** 和 **macOS 系统** 上使用常见虚拟机软件（VirtualBox / VMware / UTM / Parallels）搭建虚拟机。

---

## 🖥️ Windows 系统搭建虚拟机（以 VirtualBox 为例）

### Step 1. 下载 VirtualBox
- 前往 [VirtualBox 官网](https://www.virtualbox.org/)，下载对应的 Windows 版本安装包。

### Step 2. 安装 VirtualBox
- 打开安装包，点击“下一步”，保持默认设置即可完成安装。

### Step 3. 新建虚拟机
1. 打开 VirtualBox，点击 **“新建”**。  
2. 设置参数：  
   - **名称**：Ubuntu-Test（自定义）  
   - **类型**：Linux  
   - **版本**：Ubuntu (64-bit)  
   - **内存**：2048 MB（可根据主机内存调整）  
   - **虚拟硬盘**：20 GB（动态分配，VDI 格式）

### Step 4. 挂载系统 ISO
- 进入虚拟机 **设置 → 存储 → 光驱**，选择需要安装的 ISO 文件（如 Ubuntu 或 Windows 系统镜像）。

### Step 5. 启动并安装系统
- 点击启动虚拟机，进入 ISO 启动界面，按照安装向导完成操作系统安装。

### Step 6. 安装增强功能（可选）
- 在 VirtualBox 窗口顶部选择 **设备 → 插入增强功能光盘**，安装后可以支持：
  - 自动调整分辨率  
  - 共享剪贴板  
  - 拖拽文件  

---

## 🍎 macOS 系统搭建虚拟机（以 UTM 为例）

### Step 1. 下载 UTM
- 前往 [UTM 官网](https://mac.getutm.app/)，下载并安装 UTM（免费，基于 QEMU）。

### Step 2. 新建虚拟机
1. 打开 UTM，点击 **“新建 VM”**。  
2. 设置参数：  
   - **系统类型**：Windows / Linux  
   - **CPU**：2 核  
   - **内存**：4 GB  
   - **硬盘**：20 GB 动态分配  

### Step 3. 挂载系统镜像
- 在虚拟机设置中，选择 **CD/DVD → ISO 镜像文件**，加载 Windows 或 Ubuntu 的安装镜像。

### Step 4. 启动并安装系统
- 启动虚拟机后进入安装界面，按照提示完成系统安装。

### Step 5. 安装增强功能（可选）
- 在 UTM 菜单中选择 **安装 Spice Tools**，可以优化：
  - 显示效果  
  - 文件复制/粘贴  
  - 窗口缩放

## 🔧 小技巧
- Windows 推荐：
  - **VirtualBox**（免费，功能全）  
  - **VMware Workstation Pro**（免费，性能更强）  
- macOS 推荐：
  - **UTM**（免费）  
  - **Parallels Desktop**（付费，体验最佳）  
  - **VMware Fusion**（部分版本免费）
